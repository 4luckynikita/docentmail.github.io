Question: Write the snipped into a single line. Put semicolons to don't change the snippet logic.

{% highlight js %} x=d (f+g ).some_method() {% endhighlight %}

Tip: Recall "Immediately-Invoked Function Expression (IIFE)" JavaScript concept.


Answer: Because of "Immediately-Invoked Function Expression (IIFE)" JavaScript interpreter treats like call of function with name d and paramater f+g. Like this: {% highlight js %} x= d(f+g ).some_method(); {% endhighlight %}

There are "five problematic" characters to be careful with : "(" , "[" , "+" , "-", "/" . Never omit a semicolon before a statement beginning with the “five problematic” characters.

Just use "defensive semicolon" approach to avoid problems like this: {% highlight js %} x= d ;(f+g ).some_method() {% endhighlight %}
