Question: Give the code snippet that declares variable aVar with value 10 in local scope and not visible in Global scope

Tip: There are two types of scope in JavaScript: Global and local for functions


Answer:
Variable aVar belongs to the local scope of the function aFun: {% highlight js %} function aFun(){ var aVar=10; } {% endhighlight %}
