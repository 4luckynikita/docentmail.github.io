Question: What would be result of execution of the snippet?

{% highlight js %} function func1() { theVar="value1"; } func1(); alert(theVar); {% endhighlight %}

Tip: Pay attention that var is absent.


Answer: Alert with text value1
because of:
Statement theVar="value1"; (without var) creates variable in a global scope (of course if variable was not defined before).
