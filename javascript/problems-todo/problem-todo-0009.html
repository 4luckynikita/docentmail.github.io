Question: how interpreter works
    var a = 12+24*3-5;
Tip: consider order of operation
Answer:
    1. calculates hight priority 24*3 and converts to var a = 12+72-5;
    2. calculates the same priorities from left to right 12+72   var a = 84-5;
    3. calculates the same priorities from left to right 84-5   var a = 79;
    4. do assigment;
