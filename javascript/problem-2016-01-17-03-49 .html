Question: What would be the alert message for the code snippet?

{% highlight js %} alert(aVar); var aVar=5; {% endhighlight %}

Tip: Recall hoisting behavior of JavaScript variables


Answer: Value would be undefined.

Explanation: Because variable declarations (and declarations in general) are processed before any code is executed. The snippet above is equivalent to {% highlight js %} var aVar; alert(aVar); aVar=5; {% endhighlight %}
