Question: What would be result of execution of the snippet?

{% highlight js %} function myFunction(){ alert("Hello from myFunction "); } function testFunctionHoisting(){ alert(typeof myFunction); myFunction(); var myFunction = function() { alert("hello from local myFunction"); }; } testFunctionHoisting(); {% endhighlight %}

Tip: Inner myFunction is the variable. Recall the hoisting behavior for variables.


Answer:

  • first alert alert(typeof myFunction); shows message undefined because local myFunction variable according hoisting behavior has value "undefined".
  • Call myFunction(); throws error Fix errors in you script: TypeError: myFunction is not a function because value of local myFunction variable is "undefined" and it is not function.
