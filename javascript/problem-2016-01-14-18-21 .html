Question: Which of the names of variables are correct?

  1. 2things
  2. something4you
  3. my_variable_name
  4. aaa+bbb
  5. variable!
  6. #vvv
  7. myVariableName
  8. MyVariableName
  9. _uuu
  10. uuu_uuu
  11. ddd8kjh
  12. else
  13. MY_VARIABLE_NAME
  14. $item


Tip: Try to recall rules for JavaScript variables rules


Answer: Green is good

  1. 2things - starts from digit
  2. something4you - valid usage digit
  3. my_variable_name - underscore style
  4. aaa+bbb - "+" is used
  5. variable! - "!" is used
  6. #vvv - "#" is used
  7. myVariableName - camel style (good practice to lowercase for first letter)
  8. MyVariableName - valid but bad practice to uppercase for first letter
  9. _uuu
  10. uuu_uuu
  11. ddd8kjh
  12. else - reserved word is used
  13. MY_VARIABLE_NAME - valid but starts from uppercase (bad practice) and all uppercase style is recomended to use for constants
  14. $item - Hungarian notation. Usually using by jQuery
