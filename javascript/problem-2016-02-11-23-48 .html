Question: Why you have got error Fix errors in you script: TypeError: a is not a function for the snippet?

{% highlight js %} var a=1 var b=a (function() { alert('hello'); })(); {% endhighlight %}

Tip: Recall "Immediately-Invoked Function Expression (IIFE)" and “five problematic” characters concepts.


Answer: Because "(" is one of the "five problematic" characters semicolon would not be automatically inserter before.

The code would be equal to: {% highlight js %} var a=1; var b=a(function() { alert('hello'); } )(); {% endhighlight %} so a here is treated as function name;

There are "five problematic" characters to be careful with : "(" , "[" , "+" , "-", "/" . Never omit a semicolon before a statement beginning with the “five problematic” characters.
