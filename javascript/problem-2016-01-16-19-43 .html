Question: Give the code snippet that declares variable aVar with value 10 in Global scope

Tip: There are two types of scope in JavaScript: Global and local for functions


Answer:
this declaration out of the any function: {% highlight js %} var aVar=10; {% endhighlight %} or aVar would be in Global scope if it was mentioned first time in the statement: {% highlight js %} function aFun(){ aVar=10; } {% endhighlight %} but not this ( variable aVar declared not in Global scope but in scope of function aFun) {% highlight js %} function aFun(){ var aVar=10; } {% endhighlight %}
