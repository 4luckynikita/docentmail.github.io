Question: Is this a valid JavaScript snippet?

{% highlight js %} var foo = 10 alert(foo) {% endhighlight %}

Tip: Recall "Rules of Automatic Semicolon Insertion"


Answer: Yes, it is a valid snippet. JavaScript interpreter will automatically insert semicolons like: {% highlight js %} var foo = 10; alert(foo); {% endhighlight %} because of "Rules of Automatic Semicolon Insertion". Try original one and see "10" alert message.
