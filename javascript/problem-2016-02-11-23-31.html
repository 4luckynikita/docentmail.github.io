Question: What would be result of execution of the snippet?

{% highlight js %} var x=1, y=1; x -- y alert(x); alert(y); {% endhighlight %}

Tip: Recall "five problematic characters" concept.


Answer: First alert 1 (value of x), second alert 0 (value of y).

Explanation: There are "five problematic characters" to be careful with : "(" , "[" , "+", "-" , "/".
Never omit a semicolon before a statement beginning with the "five problematic characters".
Since -- can serve as either a prefix or a suffix, but the -- cannot be preceded by a newline.

The snippet is equal to: {% highlight js %} var x=1, y=1; x; -- y; alert(x); alert(y); {% endhighlight %}
