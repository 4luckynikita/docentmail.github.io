Question: What is result of execution of the snippet?

{% highlight js %} function a(){ return 1; } alert (a()); {% endhighlight %}

Tip: Recall "Rules of Automatic Semicolon Insertion" and "The five restricted productions"


Answer: alert with "undefined" text.

Explanation: "return" belongs to "The five restricted productions". The five restricted productions are return, throw, break, continue, and post-increment/decrement. JavaScript interpreter will automatically insert semicolon after "return" like: {% highlight js %} function a(){ return; 1; } alert (a()); {% endhighlight %}
