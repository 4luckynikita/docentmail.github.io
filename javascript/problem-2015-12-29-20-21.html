Question: replace the statement {% highlight js %} var x= "Hello", cat= "Bob", height = 20; {% endhighlight %} with multiple statements where each statement contains only one “=”. And they doing the same.

Tip: comma separator


Answer: Use semicolons instead of commas. {% highlight js %} var x= "Hello"; var cat= "Bob"; var height = 20; {% endhighlight %}
