Question: What would be alert messges for the code snippet?

{% highlight js %} alert(typeof myFunction); alert(typeof OtherFunction); function myFunction(){ alert("Hello from myFunction "); } var myOtherFunction = function() { alert("hello from local myFunction"); }; {% endhighlight %}


Tip: Recall the difference in hoisting behavior for variables and functions


Answer:

  • alert(typeof myFunction); returns function. Function gets hoisted completely (name and implementation).
  • alert(typeof OtherFunction); returns undefined. For variable gets hoisted declaration only. Value would be "undefined".
