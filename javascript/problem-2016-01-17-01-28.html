Question: What would be result of execution of the snippet?

{% highlight js %} for (var i=0; i<10; i++) { var theVar="value1"; } alert(theVar); {% endhighlight %}

Tip:Pay attention that the code block code is defined not by function.


Answer: Alert with text value1
because of:
Code block (grouped statements with curly braces { ... } ) created by if, for, while, switch and other statements does not define a new scope. Only function has it's own scope.
