Question: what does alert (x[1]); show?

{% highlight js %} var x = [1,2,3]; var y = x; y[1]=100; alert (x[1]); {% endhighlight %}

Tip: y and x are arrays. The variables contain references to the same array sitting in memory.


Answer: 100
Because variables y and x have reference type

Explanation: Try to become familiar with the distinction between primitive and reference types.
