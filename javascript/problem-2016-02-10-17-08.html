Question: Are these a valid JavaScript statements?

{% highlight js %} i ++; {% endhighlight %} {% highlight js %} i ++; {% endhighlight %}

Tip: Recall "Rules of Automatic Semicolon Insertion" and "The five restricted productions"


Answer:

{% highlight js %} i ++; // is valid {% endhighlight %} {% highlight js %} i ++; // is incorrect {% endhighlight %}

Explanation: post-increment/decrement belongs to "The five restricted productions". The five restricted productions are return, throw, break, continue, and post-increment/decrement. New line is not allowed.
