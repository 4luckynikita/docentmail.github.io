Question: What would be result of execution of the snippet?

{% highlight js %} var theVar = 'value1'; function func1() { var theVar="value1"; function func2() { alert(theVar); } func2(); } func1(); {% endhighlight %}

Tip: What about local scope for function?


Answer: Alert with text value1
because of:
value1 - belongs to Global scope.
value2 - belongs to scope of function func1.
Scope of the func2 inherit scope of func1.
